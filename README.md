# フーリエ変換 解析アプリ

StreamlitとRenderを使用したフーリエ変換Webアプリケーション 

URL → https://fouriertransformwebapplication-ecvdbth4rprkuqb9s97v79.streamlit.app/
リポジトリ内にある"y=sin(100x).csv"ファイルをアプリ内で読み込ませることでw = 100でピークのある関数が出力されます


## 機能
- CSVファイルのアップロード
- リアルタイムフーリエ変換計算
- 結果の可視化とダウンロード
- サンプルデータ生成

## 使用技術
- Python 3.9+
- Streamlit
- NumPy
- Matplotlib
- Pandas
